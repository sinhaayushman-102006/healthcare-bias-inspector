## 🎯 Problem Statement

AI models in healthcare can unintentionally introduce **bias** against certain groups (e.g., age, gender), leading to:

- ❌ Incorrect diagnoses  
- ❌ Unequal treatment  
- ❌ Reduced trust in AI systems  

---

## 💡 Solution

**Healthcare AI Bias Inspector** is a smart system that:

- 🔍 Detects bias in ML models  
- 🧠 Explains model decisions using SHAP  
- ⚖️ Measures fairness across sensitive groups  
- 🛠️ Applies bias mitigation techniques  
- 📄 Generates downloadable fairness reports  

---

## 🚀 Features

### 📊 Data Analysis
- Upload your dataset or use built-in datasets  
- Preview and explore data instantly  

### 🧠 Bias Detection
- Calculates accuracy per group  
- Identifies bias gap across sensitive attributes  

### 🔍 Explainability (SHAP)
- Visualizes feature importance  
- Understands model behavior  

### 🛠️ Bias Mitigation
- Applies fairness improvement techniques  
- Compares before vs after results  

### 📄 PDF Report
- Download a structured bias analysis report  

### 🤖 AI Assistant
- Ask questions about model performance and fairness  

### 🎮 Interactive UI
- Animated dashboard  
- Interactive charts  
- Modern SaaS-style design
